
# Mediapipe Handtracking




## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install packages

```bash
  pip install opencv-python
  pip install mediapipe
```

Start the program

```bash
  python ./Hand.py
```

Stop the program

```bash
  `Ctrl + C` or 'Ctrl + Shift + C `
```
